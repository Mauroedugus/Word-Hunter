# Word Hunter

Um jogo web divertido com foco no publico infantil, onde são exibidos palavaras e imagens para que sejam relacionadas.

## Tecnologias
- Node.js  
- Express  
- HTML5 / CSS3 / JavaScript  
- NPM para gerenciamento de dependências

## Instalação
Clone o repositório e instale as dependências:
```bash
git clone https://github.com/Mauroedugus/Word-Hunter.git
cd Word-Hunter
npm install
```
Para rodar o jogo basta inserir no terminal o comando e acessar a porta no navegador:
```bash
npm start
